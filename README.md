# TradeOS | Field Service Management

TradeOS is a white-label platform for reliability and efficiency in the trades.

## 🚀 Getting Started

To understand the project architecture and how to contribute (manually or via AI agents), please refer to the documentation:

- **[Agentic Orchestration Guide](./docs/README.md):** Technical details on the stack, multi-tenancy, and coding patterns.
- **[Original Vision](./docs/original_prompt.md):** The core product prompt and goals.
- **[Database Schema](./docs/backend.json):** Definition of entities and Firestore paths.

## 🛠 Development

```bash
npm run dev
```

The application is split into two main portals:
- `/dispatcher`: For office management.
- `/technician`: For field workers.
